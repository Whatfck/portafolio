# Portafolio - Daniel Perez

Portafolio profesional construido con Next.js 14, TypeScript y CSS global.

## Desarrollo local

```bash
npm install
npm run dev
```

Abrir en http://localhost:3000.

## Scripts

- npm run dev
- npm run build
- npm run start
- npm run lint

## Estructura actual

```txt
app/[lang]/layout.tsx
app/[lang]/page.tsx
components/
lib/dictionaries.ts
styles/globals.css
middleware.ts
public/
```

## Personalización

- Contenido multilenguaje: lib/dictionaries.ts
- Secciones UI: components/
- Estilos globales: styles/globals.css
